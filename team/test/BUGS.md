# Bug 列表

## Bug #1: 图层管理功能缺失

**严重程度:** 🟡 中  
**模块:** Layers 面板

**描述:**  
图层面板只能显示和选中图层，缺少基本的管理功能。

**缺失功能:**
- 添加新图层
- 删除图层
- 重命名图层
- 图层排序/拖拽

**建议:** 参考 Figma 的图层面板，添加右键菜单或工具栏按钮。

---

## Bug #2: Inspector 输入框响应慢

**严重程度:** 🟡 中  
**模块:** Inspector 面板

**描述:**  
点击 Name 输入框后，输入操作超时（>8秒无响应）。

**复现步骤:**
1. 选中 Card 图层
2. 点击 Name 输入框
3. 尝试输入文字
4. 输入无响应或超时

**建议:** 检查输入框的事件绑定和状态管理。

---

## Bug #3: "+ Add" 按钮无响应

**严重程度:** 🟡 中  
**模块:** Keyframes 区域

**描述:**  
点击 "+ Add" 按钮添加关键帧时，按钮点击超时。

**建议:** 检查按钮的 onClick 事件是否正确绑定。
