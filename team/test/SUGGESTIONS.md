# 改进建议

## 优先级高 🔴

### 1. 完善图层管理
- 添加「新建图层」按钮
- 支持右键菜单（重命名/删除/复制）
- 支持拖拽排序

### 2. 修复输入框交互
- 检查 React 状态更新逻辑
- 确保输入框能正常接收键盘事件

## 优先级中 🟡

### 3. 丰富 Inspector 属性
当前只有 Name，建议添加：
- 位置 (x, y)
- 大小 (width, height)
- 透明度 (opacity)
- 颜色 (background)

### 4. 添加关键帧功能
- "+ Add" 按钮需要能正常工作
- 支持删除关键帧
- 支持重命名关键帧

---

## 优先级低 🟢

### 5. 交互体验优化
- 添加键盘快捷键
- 添加 hover 状态提示
- 添加操作反馈动画

### 6. 可访问性
- 添加 ARIA 标签
- 支持键盘导航
