# Toumo Design System

> State Machine Motion Design Tool 的视觉设计规范

## 设计理念

**"专业、克制、流畅"**

Toumo 是一个动效设计工具，界面本身就应该是动效品质的最佳展示。我们追求：

1. **深邃的专业感** - 深色主题，让用户专注于创作
2. **精确的层次感** - 清晰的信息架构，一目了然
3. **微妙的生命力** - 界面动效体现工具的核心价值

## 核心原则

### 1. 深色优先
- 主背景 `#0a0a0b` - 近乎纯黑，专业感
- 避免纯黑 `#000` - 太死板
- 避免过亮 - 伤眼睛

### 2. 层次分明
- 用微妙的明度差异区分层级
- 边框用于分隔，不用于装饰
- 阴影极少使用，保持扁平

### 3. 动效即品牌
- 所有交互都有动效反馈
- 动效时长 150-300ms
- 曲线优先 ease-out

### 4. 信息密度
- 紧凑但不拥挤
- 留白有节奏
- 重要信息突出
